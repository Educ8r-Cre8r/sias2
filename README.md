1. **Clone or download this repository**

   ```bash

   npm install
   ```

3. **Set up API key** (for content generation):
   ```bash
   export ANTHROPIC_API_KEY="sk-ant-your-key-here"
   ```

   ```
   ANTHROPIC_API_KEY=sk-ant-your-key-here
   ```

4. **Open the website**:
   Simply open `index.html` in your web browser, or use a local server:
   ```bash
   # Python 3
   python3 -m http.server 8000

   # Then visit http://localhost:8000
   ```

## ğŸ”§ Usage

### For Teachers (Viewing Content)

1. Open `index.html` in a web browser
2. Click "Explore Gallery" to scroll down
3. Use category filters to browse by science type
4. Click the ğŸ““ notebook icon on any photo to view educational content
5. Read AI-generated lesson materials, tips, and resources
6. Click NGSS links to explore standards

### For Administrators (Managing Content)

#### Step 1: Organize Existing Images

If you have images in the main `/images/` folder that need categorizing:

```bash
cd tools

# Auto-categorize images using Claude AI
node categorize-images.js

# Review results in categorization-results.json
# Edit any incorrect categorizations

# Move images to category folders
node organize-images.js
```

This creates `gallery-metadata.json` which powers the gallery.

#### Step 2: Generate Educational Content

```bash
# Generate content for all images
node generate-content.js --all

# Or generate for specific category
node generate-content.js --category life-science

# Or generate for single image
node generate-content.js --image bee.jpeg
```

â±ï¸ **Note**: Generating content for 72 images takes approximately 1-2 hours due to API rate limits.

#### Step 3: Add New Photos

1. Place new photo in appropriate category folder:
   ```
   images/life-science/butterfly.jpg
   ```

2. Generate content:
   ```bash
   node generate-content.js --image butterfly.jpg
   ```

3. Update `gallery-metadata.json` manually or re-run `organize-images.js`

## ğŸ› Troubleshooting

### Images not loading

**Issue**: Gallery shows "Image not found" placeholders

**Solution**:
1. Check that images exist in `/images/{category}/` folders
2. Verify paths in `gallery-metadata.json`
3. Check browser console for 404 errors

### Educational content not loading

**Issue**: Modal shows "Content not available"

**Solution**:
1. Check that JSON files exist in `/content/{category}/` folders
2. Verify `contentFile` paths in `gallery-metadata.json`
3. Ensure JSON files are valid (use JSONLint.com)

### Parallax effect not working on mobile

**Solution**: This is intentional. `background-attachment: fixed` doesn't work on iOS Safari, so it gracefully degrades to a static background.

### NGSS links not working

**Issue**: Links go to 404 pages

**Solution**:
1. Check that NGSS codes are formatted correctly
2. Verify the `generateNGSSUrl()` function in `script.js`
3. NGSS website structure may have changed - update URLs as needed

### API errors during content generation

**Issue**: "Error: 401 Unauthorized"

**Solution**:
1. Check that `ANTHROPIC_API_KEY` environment variable is set
2. Verify API key is valid at console.anthropic.com
3. Check API usage limits and billing
